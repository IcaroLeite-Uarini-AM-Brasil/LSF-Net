## 💡 Recursos Rápidos

### Setup em 5 minutos
```bash
git clone https://github.com/IcaroLeite-Uarini-AM-Brasil/LSF-Net.git
cd LSF-Net
bash scripts/setup.sh
python backend/app.py
```

### Acessar Aplicação
```
http://localhost:5000
```

### Verificar Status da API
```bash
curl http://localhost:5000/api/health
```
